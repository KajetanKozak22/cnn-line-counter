# CNN Line Counter & Predictor

A convolutional neural network (CNN) based application designed for image analysis and counting lines on white surfaces. It utilizes a trained PyTorch model and features a graphical user interface (GUI).

## Features

* **Automated Prediction** – Line recognition and counting based on the selected image file.
* **Real-Time Preprocessing** – Built-in median filter that automatically removes noise from the image.
* **Simple UI** – Intuitive interface displaying the image preview, network prediction, and the ground truth value from a CSV database.
* **Separate Training Module** – Ability to train the network on your own custom dataset.

## Requirements

## How to Run the Project

### Prerequisites
* Python 3.8+ and pip installed.
* The entire repository downloaded (including the `dataset` folder).

### Steps

```text
# 1. Clone or download the repository
git clone [--------------------------------------------]

# 2. Install the required packages
pip install torch pandas opencv-python pillow numpy

# 3. (Optional) Train your own model 
python uczenie.py

# 4. Run the GUI application
python program.py
```

### First Run
1. After launching the application (`python program.py`), click the green **"Wybierz obrazek (.png)"** (Select Image) button.
2. Select any file from the `dataset` folder.
3. The system will process the image, and the interface will update with the preview, filename, and prediction values.
4. You will see the **"PRAWDA"** (TRUE) value (fetched from the CSV) and the **"WYNIK"** (RESULT) value (calculated by the network) displayed on the screen.
